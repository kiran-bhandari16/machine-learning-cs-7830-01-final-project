# Fuzzy C-Means Clustering with Hyperparameter Tuning

This project implements **Fuzzy C-Means (FCM)** clustering and evaluates how different hyperparameters affect clustering performance. The goal is to run the model with different parameter settings, compare the results using clustering evaluation metrics, and identify which combination gives the **best score**.

## What is Fuzzy C-Means?

Fuzzy C-Means is a soft clustering algorithm. Unlike K-Means, where each data point belongs to exactly one cluster, Fuzzy C-Means assigns each point a **membership value** for every cluster.

This means:
- A point can partially belong to multiple clusters
- Membership values are between 0 and 1
- The sum of membership values for each point is 1

This is useful when cluster boundaries are not very clear.

## Project Objective

The main objectives of this project are:

- Apply **Fuzzy C-Means clustering**
- Tune important hyperparameters
- Evaluate clustering quality
- Find the best parameter combination based on clustering scores
